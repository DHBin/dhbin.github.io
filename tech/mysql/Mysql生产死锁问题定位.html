# 删除数据
delete `db`.`table` where org_code = ?

# 新增数据
insert into `db`.`table` values(...)

原因：org_code这个字段上存在索引，RC事务级别会产生间隙锁把相邻的位置锁住，多条消息过来多线程消费导致锁相互持有最终导致死锁

解决方法: 在业务允许的情况下，减低mysql事务隔离级别到RR
