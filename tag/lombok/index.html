    注解

    @AllArgsConstructor

    作用

    生成包含所有字段的构造器

    参数

    • staticName : 不为空的话，生成一个静态方法返回实例，并把构造器设置为private
    @AllArgsConstructor(staticName = "create")
public class Example {

    private int foo;
    private final String bar;
}
